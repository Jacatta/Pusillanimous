Abstract

I created this project to showcase my mobile development abilities. I wanted to create a quick prototype so that I could get feedback and change things accordingly. I started with the type of gameplay I wanted, so in Unity I build basic game mechanics and tested with users. After seeings that people liked the type of game. I then began drawing out the UI design, and creating the necessary assets in Photoshop. I am still in the process of bringing all aspects together, but after 3 weeks of work this is what I have to show.

This was the initial concept of the type of game I wanted to create. After seeing the addictive reaction towards the game, I decided to move forward with the idea of a quick tapping game.
